# MoodBoost

An engaging web app that gamifies emotional wellness tracking. Users log their daily mood through an interactive emoji interface, earn streaks and rewards for consistency, and get personalized insights. Features include:

- Interactive mood logging with animated emojis
- Daily streak tracking with rewards
- Weekly/monthly mood patterns visualization
- Shareable mood cards for social media
- Progressive achievement system

## Tech Stack
- HTML5/CSS3 with animations
- Vanilla JavaScript for interactivity
- LocalStorage for data persistence
- Chart.js for mood visualizations
- Cypress for E2E testing
